INSTA_TESTING_BOT
=================
Fast-response skeleton for an Instagram+Telegram+AI hybrid bot.

This package contains:
- bot_code/: Minimal Python skeleton (handlers, config loader)
- data/: folders for user files and media
- configs/: example config files (DO NOT put real credentials here)
- docs/: documentation and messages
- messages.json: fast-response messages used by the bot

IMPORTANT: This skeleton contains placeholders only. Replace tokens, API keys,
and implement real API calls (instagrapi, python-telegram-bot, OpenAI/Gemini clients)
before running.
